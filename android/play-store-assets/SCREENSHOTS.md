# Screenshot Guidelines

## Recommended Screenshots (in order)

### 1. Main Translation Screen

**Caption**: "Translate into multiple languages simultaneously"

- Show the main translation screen with text input
- Display 3-4 translation results in different languages
- Include romanization for at least one non-Latin script
- Show the language selection and settings icons

### 2. Image Translation

**Caption**: "Extract and translate text from images"

- Show an image with detected text regions highlighted
- Display the translation bottom sheet with results
- Demonstrate the "Translate All" button

### 3. Translation History

**Caption**: "Access your translation history anytime"

- Show the history screen with several saved translations
- Display the search bar
- Show favorite/delete actions on history items

### 4. Language Selection

**Caption**: "Choose from 59+ languages"

- Show the language selection bottom sheet
- Display multiple languages with their native names
- Show selected languages highlighted

### 5. Romanization Feature

**Caption**: "Get phonetic romanization for non-Latin scripts"

- Show a translation with romanization visible
- Use examples like Japanese, Chinese, Arabic, or Russian
- Highlight the romanization toggle

## Screenshot Specifications

### Phone Screenshots

- **Resolution**: 1080 x 1920 px (minimum)
- **Aspect Ratio**: 9:16
- **Format**: PNG (24-bit, no alpha)
- **Color Space**: sRGB

### Tablet Screenshots (Optional)

- **Resolution**: 1200 x 1920 px or 1600 x 2560 px
- **Aspect Ratio**: 10:16
- **Format**: PNG (24-bit, no alpha)

## Tips for Great Screenshots

1. **Use Real Content**: Show actual translations, not placeholder text
2. **Clean UI**: Hide system UI elements (status bar, navigation bar) or use clean mockups
3. **Consistent Theme**: Use the same theme (light or dark) across all screenshots
4. **Readable Text**: Ensure all text is legible at thumbnail size
5. **Feature Highlights**: Each screenshot should highlight one key feature
6. **Localization**: Consider creating localized screenshots for major markets

## Tools

- **Screenshot Tool**: Android Studio Device Manager or physical device
- **Mockup Generator**: Use tools like Screely, Mockuphone, or Device Art Generator
- **Image Editor**: Use Figma, Photoshop, or GIMP for final touches
- **Frame Generator**: Google's Device Art Generator or Appure

## Example Translation Text

Use diverse, interesting examples:

- **English to Spanish/French/German**: "Hello, how are you today?"
- **English to Japanese/Chinese/Korean**: "Where is the nearest train station?"
- **English to Arabic/Hebrew**: "Thank you very much"
- **English to Hindi/Thai**: "I would like to order food"
- **English to Russian/Ukrainian**: "What time is it?"

## Accessibility

- Ensure screenshots demonstrate accessibility features
- Show high contrast where applicable
- Include examples of TalkBack-friendly UI elements
